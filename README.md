MiniMessenger

Real-time messaging application built with FastAPI, React, and WebSockets.
The project demonstrates a modern full-stack architecture with cloud deployment and containerization.

Live Demo:
Frontend → https://mini-messenger-scib.vercel.app
------------------------------------
Features

User registration and authentication

Real-time messaging using WebSockets

Conversation system between users

Search users by username

Message history storage

Dockerized backend and frontend

Cloud deployment (Render + Vercel)
